    • Réponse

      Puisque nous avons les longueurs des 3 côtés du triangle alors on va utiliser la réciproque du théorème de Pythagore qui permet de tester si un triangle est rectangle ou pas.

      Pour rappel, dans un triangle rectangle l'hypoténuse est le côté en face de l'angle droit et c'est le plus grand.

      Utilisons les données de l'énoncé pour faire les calculs :

      \(AB^2 + BC^2 = 3^2 + 4^2 = 9 + 16 = 25 = 5^2 = CA^2\)

      D'après la réciproque de Pythagore le triangle est rectangle en B d'hypoténuse CA.

      Protocole de construction du triangle ABC

      Nom Définition Valeur
      Point A   A = (-8, -4)
      Point B   B = (-5, -4)
      Point C   C = (-5, 0)
      Triangle t1 Polygone(A, B, C) t1 = 6
      Segment c Segment(A, B, t1) c = 3
      Segment a Segment(B, C, t1) a = 4
      Segment b Segment(C, A, t1) b = 5

    • Réponse

      L’angle \(\widehat{CAB}\) a pour côté adjacent [AB] ainsi : \(\cos(\widehat{CAB}) = \dfrac{AB}{CA} = \dfrac{3}{5} = 0,6\).

      En utilisant une calculatrice ou Geogebra on appuie sur la touche \(\arccos\) ou \(\cos^{-1}\) pour obtenir l'angle à partir de la valeur du cosinus.

      On obtient \(\widehat{CAB} \simeq 53,1301\) et nous voulons un arrondi par défaut au centième donc on garde uniquement les 2 premières décimales.

      Finalement, \(\boxed{\widehat{CAB} \simeq 53,13}\) degrés.
